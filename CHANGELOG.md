# CHANGELOG

## 2026-03-31（doctool）

### New Features

- 指摘分類のカテゴリ数を可変化（設定シートのみで変更可能、デフォルトは9カテゴリ）
- カテゴリ変更検出エラー（E13）の追加：前回と異なるカテゴリ設定での実行を検出して処理中断
- カテゴリ未登録エラー（E12）の追加：設定シートが空の場合に処理中断

### Improvements

- **保守性の向上**: カテゴリ追加・削除時に VBA コード修正不要。`[ツール本体] 指摘分類マッピング設定` シートのみで管理できる
- **拡張性の確保**: 将来的に最大 676 種類まで対応可能（エイリアス a〜zz）
- **既存データとの互換性**: 既存の 9 種類（a〜i）はそのまま利用可能。レビュー記録一覧の既存データに影響なし
