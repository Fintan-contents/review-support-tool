# エラーハンドリング・メッセージ一覧

---

## 同じレビュー回数での重複実行

**状況**:

- レビュー回数 = 1 で既にレビュー結果シート「レビュー結果 1 回目」が存在
- 再度レビュー回数 = 1 で実行

**動作**:

確認ダイアログは表示されず、以下を自動的に上書きする。

1. **結果シート**: `Application.DisplayAlerts = False` で既存シートを削除し、新規生成する
2. **レビュー記録票（指摘一覧）**: 同回数の既存行を上書き更新する
3. **レビュー記録一覧**: 同文書ID・同回数の既存行を上書き更新する（新規行は追加しない）

## エラー検出と処理

### レビュー日時情報のエラー

**検出条件**:

- コメントの要素数が 5 未満（名前、実施日、開始時刻、終了時刻、レビュー時間が入っていない）

**エラー処理**:

1. エラーシートテンプレートをコピー
2. エラーシート名: `エラーシート`
3. エラー内容:「レビュー日時コメントが正しく入力されていません。実施日、開始時刻、終了時刻、レビュー時間を全て入力してください。」
4. 場所へのハイパーリンク作成
5. 処理は続行（エラーがあっても他の指摘は処理する）

### レビュー記録票が見つからない

**検出条件**:

- 対象設計書に対応するレビュー記録票（ファイル名に「レビュー記録票」を含む）が開いていない

**エラー処理**:

- メッセージボックス表示
- その設計書の処理をスキップ

### シートが存在しない

**検出条件**:

- レビュー記録一覧の指定シートが存在しない
- レビュー記録票の指定シートが存在しない

**エラー処理**:

- メッセージボックス表示
- 処理を中断
