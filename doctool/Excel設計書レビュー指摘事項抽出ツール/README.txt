1. ツールの概要
─────────────────────────────────────────

本ツールは、Excel 設計書に記載されたレビュー指摘事項（Excelコメント）を
自動的に抽出し、レビュー記録票へ転記するマクロツールです。

主な機能:
  ・レビュー指摘の一括抽出（複数の設計書ファイルを一度に処理）
  ・レビュー記録票への自動転記
  ・レビュー記録一覧（集計シート）への自動集計
  ・レビュー結果シートの自動生成
  ・不要なコメント・結果シートの一括削除


─────────────────────────────────────────
2. 前提条件・準備
─────────────────────────────────────────

【動作環境】
  ・Microsoft Excel（Windows版）が必要です。
  ・マクロの実行を有効にして開いてください。

【ファイルを開く前の準備】
  (1) 「Excel設計書レビュー指摘事項抽出ツール.xlsm」を開きます。
  (2) セキュリティの警告が表示された場合は「コンテンツを有効にする」を
      クリックしてください。
  (3) 処理対象の設計書ファイル（.xlsx）を全て開いた状態にしてください。
  (4) レビュー記録票ファイル（.xlsx）も開いた状態にしてください。
  (5) 必要に応じてレビュー記録一覧ファイルも開いておきます。

【設計書ファイルの命名規則】
  設計書ファイルは「基本設定」シートで指定したファイル名パターンに
  一致する必要があります。パターンに一致しないファイルは処理対象外です。


─────────────────────────────────────────
3. 使用手順
─────────────────────────────────────────

◆ Step 1: 基本設定の確認（初回のみ）

  ツールを開くと「基本設定」シートが表示されます。
  以下の項目を確認・入力してください:

  ・レビュー記録一覧ファイルパス（任意）
      → レビュー記録一覧に集計する場合のみ入力
  ・レビュー記録一覧シート名（使用する場合）
  ・工程、レビュア、レビュイの参照設定
      → 固定値を直接入力するか、設計書のシート名+セル番地を指定

◆ Step 2: 指摘分類の確認（初回または変更時のみ）

  「指摘分類マッピング設定」シートで指摘分類（カテゴリ）の一覧を確認します。
  カテゴリの追加・変更が必要な場合はこのシートを編集してください。

  注意: レビューの途中でカテゴリを変更すると、処理が中断されます（後述）。

◆ Step 3: レビュー回数の入力

  「基本設定」シートの「レビュー回数」欄に今回のレビュー実施回数を入力します。
  （例: 1回目のレビューなら「1」を入力）

◆ Step 4: 指摘事項の抽出実行

  「抽出」ボタンをクリックします。

  処理の流れ:
  (1) 開いているExcelファイルのうち、対象パターンに一致するファイルを検索
  (2) 各ファイルについて確認ダイアログが表示されます（C02）
      ・「はい」 → そのファイルを処理
      ・「いいえ」 → そのファイルをスキップ
      ・「キャンセル」 → 以降のファイルを全てスキップ
  (3) 処理が完了すると完了通知が表示されます（I01）

  処理結果:
  ・各設計書ファイルに「レビュー結果 N 回目」シートが生成されます
  ・レビュー記録票の対応するシートに指摘が転記されます
  ・レビュー記録一覧が設定されている場合は集計行が更新されます

◆ Step 5: 結果の確認

  各設計書ファイルの「レビュー結果 N 回目」シートで抽出内容を確認します。
  エラーがあった場合は「エラーシート」に記録されます（ツール本体シート）。

◆ コメント・シートの削除（任意）

  ・「コメント削除」ボタン: 設計書のコメントを一括削除
  ・「レビュー結果シート削除」ボタン: 設計書のレビュー結果シートを一括削除

  どちらも対象ファイルごとに確認ダイアログ（C04 / C05）が表示されます。


─────────────────────────────────────────
4. エラーメッセージと対処法
─────────────────────────────────────────

【処理が中断されるエラー】

  E01: レビュー記録一覧のファイルパスが未入力です。
  → 「基本設定」シートのファイルパス欄を入力してください。
     レビュー記録一覧を使用しない場合は使用設定をオフにしてください。

  E02: 一覧への反映値が未入力です。
  → 「基本設定」シートの工程・レビュア・レビュイの設定を確認してください。
     固定値か、シート名+セル番地のどちらかを指定してください。

  E03: レビュー記録一覧のシート名が未入力です。
  → 「基本設定」シートのシート名欄を入力してください。

  E04: レビュー記録一覧ファイルが存在しません。
  → 指定したパスにファイルがあるか確認してください。
     ファイルを先に開くか、正しいパスを入力してください。

  E05: レビュー記録一覧ファイルに指定されたシートが存在しません。
  → レビュー記録一覧ファイルのシート名を確認してください。

  E06: レビュー記録票に〔シート名〕シートが存在しません。
  → レビュー記録票ファイルに必要なシートがあるか確認してください。

  E07: 指定されたレビュー回数が正しくありません。
  → レビュー回数が連続していない可能性があります（例: 2回目を飛ばして3回目）。
     「基本設定」シートの「レビュー回数」を確認してください。

  E08: レビュー対象文書が見つかりません。
  → 対象の設計書ファイルが開いているか確認してください。
     ファイル名が「基本設定」シートの命名パターンに一致しているか確認してください。

  E09: メモ削除対象のファイルはありませんでした。
  → コメントを持つ設計書ファイルが開いているか確認してください。

  E10: レビュー結果シート削除対象のファイルはありませんでした。
  → レビュー結果シートを持つ設計書ファイルが開いているか確認してください。

  E11: 処理中にエラーが発生しました。〔エラー番号〕〔エラー内容〕
  → 予期しないエラーです。Excelの状態（ファイルの保護、権限等）を確認してください。
     解決しない場合は担当者に連絡してください。

  E12: 指摘分類マッピング設定シートにカテゴリが登録されていません。
  → 「指摘分類マッピング設定」シートに少なくとも1件のカテゴリを登録してください。

  E13: カテゴリ設定が変更されました。
  → レビュー実施途中でカテゴリ設定が変更されたため処理を中断しました。
     対処方法:
       1. 設計書ファイル内の「レビュー結果 N 回目」シートを全て削除する
       2. レビュー回数を「1」に戻して最初から実行する
     ※ レビュー記録一覧ファイルの削除は不要です。

【警告（処理は続行）】

  W01: 対応するレビュー記録票が開いていません。
  → 対象の設計書に対応するレビュー記録票を開いてください。
     開いていない設計書はスキップされます。

  W02〜W04: 工程・レビュア・レビュイを取得できません。
  → 「基本設定」シートの参照設定（シート名・セル番地）を確認してください。
     処理は続行し、該当欄は空白で登録されます。

  W05: レビュー日時コメントが正しく入力されていません。
  → 「*」で始まるコメント（レビュー日時記録）の形式を確認してください。
     実施日・開始時刻・終了時刻・レビュー時間の全項目が必要です。

  W06: 未登録のカテゴリが使用されています。
  → コメントに使われている指摘分類の記号が「指摘分類マッピング設定」に
     登録されていません。設定シートを確認・更新してください。


─────────────────────────────────────────
5. 注意事項
─────────────────────────────────────────

  ・レビューの途中でカテゴリ設定を変更しないでください。
    カテゴリ変更が必要な場合は、既存のレビュー結果シートを全て削除してから
    1回目のレビューをやり直してください。

  ・同じレビュー回数で再実行した場合、確認なく上書きされます。
    誤って実行した場合は、設計書のレビュー結果シートを削除して
    再度正しい回数で実行してください。

  ・レビュー記録一覧ファイルは処理前に自動的に再オープン（保存→閉じる→開く）
    される場合があります（C01）。編集中の変更内容は保存されますが、
    必要に応じて事前に保存・確認してください。

  ・本ツールはWindows版Excelで動作します。Mac版では動作しません。
