================================================================================
Excel設計書レビュー指摘事項抽出ツール  ユーザーガイド
================================================================================

■ ツールの概要

このツールは、Excelレビューコメント（メモ）から指摘事項を自動的に抽出し、
レビュー記録票・レビュー記録一覧に転記するVBAツールです。

手作業で行っていた指摘事項の転記作業を自動化することで、作業時間を削減し、
転記ミスを防止します。

■ 前提条件

・Excel 2016 以降（Microsoft 365 含む）
・マクロ（VBA）の実行を許可する設定が必要です
  [ファイル] → [オプション] → [セキュリティセンター] → [セキュリティセンターの設定]
  → [マクロの設定] → 「すべてのマクロを有効にする」を選択してください
・抽出対象の設計書Excelファイルを事前に開いておく必要があります

■ ファイル構成

・Excel設計書レビュー指摘事項抽出ツール.xlsm  ← ツール本体
・レビュー記録票（別途用意）
・レビュー記録一覧（別途用意、任意）

================================================================================
■ 基本的な使用手順
================================================================================

【手順1】ツールを開く
  「Excel設計書レビュー指摘事項抽出ツール.xlsm」を開きます。
  マクロ有効化のダイアログが表示された場合は「コンテンツの有効化」を
  クリックしてください。

【手順2】対象ファイルを開く
  レビュー対象の設計書Excelファイルを開きます（複数可）。

【手順3】基本設定シートを確認する
  「基本設定」シートを開き、以下の項目を確認・設定してください。
  ・レビュー記録一覧のファイルパス（使用する場合）
  ・レビュー記録一覧のシート名（使用する場合）
  ・工程・レビュア・レビュイの取得設定

【手順4】レビュー回数を設定する
  「基本設定」シートでレビュー回数（1、2、3...）を入力します。
  初回は「1」を入力してください。
  ※必ず1から順番に実行してください。

【手順5】抽出を実行する
  「指摘事項抽出」ボタンをクリックします。
  対象設計書ファイルが見つかるたびに確認ダイアログが表示されます。
  ・「はい」     : そのファイルを抽出対象とする
  ・「いいえ」   : そのファイルをスキップする
  ・「キャンセル」: 以降のファイルをすべてスキップする

【手順6】結果を確認する
  処理完了後、各設計書Excelに「レビュー結果 N 回目」シートが生成されます。
  レビュー記録一覧を使用している場合は自動的に転記されます。

================================================================================
■ その他の機能
================================================================================

【メモ削除】
  レビュー済みのメモ（コメント）をまとめて削除します。
  「メモ削除」ボタンをクリックし、対象ファイルごとに確認ダイアログが表示されます。

【レビュー結果シート削除】
  生成されたレビュー結果シートをまとめて削除します。
  「レビュー結果シート削除」ボタンをクリックし、対象ファイルごとに確認します。

================================================================================
■ よくあるエラーと対処法
================================================================================

【E01】レビュー記録一覧のファイルパスが未入力です。
  → 「基本設定」シートでファイルパスを入力してください。
     使用しない場合は「レビュー記録一覧使用」をOFFに設定してください。

【E04】レビュー記録一覧ファイルが存在しません。
  → 設定したパスにファイルが存在するか確認してください。
     パスの区切り文字は「\」（バックスラッシュ）を使用してください。

【E08】レビュー対象文書が見つかりません。
  → 対象の設計書Excelファイルが開かれているか確認してください。
     ファイル名が「基本設定」シートの「対象ファイル名パターン」に
     一致しているか確認してください。

【E12】指摘分類マッピング設定シートにカテゴリが登録されていません。
  → 「指摘分類マッピング設定」シートにカテゴリを1件以上登録してください。

【E13】カテゴリ設定が変更されました。
  → 指摘分類マッピング設定のカテゴリ構成が、既存のレビュー結果シートと
     一致しない場合に発生します。
     【対処方法】
     1. 設計書Excelの既存のレビュー結果シート（「レビュー結果 1 回目」等、
        全て）を削除する
     2. 1回目のレビューから実行をやり直す
     ※レビュー記録一覧の削除は不要です。

【W01】レビュー記録票が開いていません。
  → 対応するレビュー記録票ファイルをExcelで開いてから再実行してください。

【W05】レビュー日時コメントが正しく入力されていません。
  → 処理は続行されます。生成された「エラーシート」を確認し、
     該当のコメント（「*」で始まる日時コメント）を正しく入力し直してください。
     形式：* 実施日 開始時刻 終了時刻 レビュー時間

【W06】未登録のカテゴリが使用されています。
  → 処理は続行されます。生成された「エラーシート」を確認し、
     該当のコメントのカテゴリ記号を「指摘分類マッピング設定」シートに
     登録されている記号に修正してください。

================================================================================
■ 注意事項
================================================================================

・レビュー回数は必ず1から順番に実行してください。前の回が存在しない状態で
  後の回を実行するとエラー（E07）が発生します。

・同じレビュー回数で再実行した場合、確認なしで既存のレビュー結果シートが
  上書きされます。

・指摘分類マッピング設定シートのカテゴリを変更する場合は、設計書Excelの
  既存のレビュー結果シートを全て削除してから1回目のレビューをやり直して
  ください（E13参照）。

・このツールはExcel VBAで動作します。処理中はExcelの操作ができなくなる
  場合があります。処理が完了するまでお待ちください。

・詳細な技術仕様は docs/ フォルダ内のドキュメントを参照してください。
