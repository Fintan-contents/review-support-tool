## テスト実施手順（概要）

1. Windows 環境で `Excel設計書レビュー指摘事項抽出ツール.xlsm` を開く
2. 設定シートで処理対象フォルダをシナリオフォルダに指定する
3. 各シナリオの手順に従って操作・実行する（詳細は「シナリオ詳細」セクション参照）
4. 期待結果と一致していることを確認する
